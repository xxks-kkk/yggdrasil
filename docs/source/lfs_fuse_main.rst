lfs\_fuse\_main module
======================

.. automodule:: lfs_fuse_main
    :members:
    :undoc-members:
    :show-inheritance:

``lfs_fuse_main.py`` is the main entrance to the LFS system implementation mentioned
in paper section 2.2. It will link to the :ref:`lfs_fuse.pyx <lfs-fuse.rst>`,
which the actual main function of the LFS system. In other words, ``lfs_fuse_main.py`` is just a wrapper.
